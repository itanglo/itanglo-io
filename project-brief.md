# Project Brief

My current domain name is expiring in a couple of days. I wasn't really happy with the website anyway, so this is a good opportunity to rebuild it.

## Audience

The website will probably be visited by potential employers or clients, peers in the web development industry and friends and family. Almost forgot, I'm also going to be a user of the website. In all likelihood the most frequent one. I would like to keep this as a place to collect links and content I come across and to talk about any side projects I work on.
